# Privacy Policy

Effective date: 2026-06-28

ChatGPT Temporary Chat Auto is an unofficial Chrome extension that helps open new ChatGPT chats with Temporary Chat enabled.

## Data Collection

This extension does not collect, transmit, sell, or share personal data.

The extension does not send ChatGPT conversation content, account information, prompts, responses, browsing history, analytics, or identifiers to any external server.

## Local Extension Settings

The extension stores only user-selected settings such as whether automatic Temporary Chat mode is enabled. These settings are saved with Chrome extension storage so they can persist between browser sessions.

## Host Access

The extension runs only on:

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`

This access is used only to apply Temporary Chat behavior on ChatGPT pages.

## Remote Code

The extension does not load or execute remotely hosted code.

## Third-Party Services

The extension does not integrate with third-party analytics, advertising, tracking, or telemetry services.

## Contact

For privacy questions, open an issue in the GitHub repository for this extension.

## Affiliation

This extension is not affiliated with OpenAI.
